# 面試準備

## 平台
 
- LeetCode
- HackerRank

## 量化讀書計畫

- 用 2 個小時解 5 題
  - Push Harder - 用更短的時間，例如 1 小時 45 分解 5 題

## 書目

- [Cracking the Coding Interview](http://www.crackingthecodinginterview.com/)
  - 這應該是必讀了
